OVERVIEW

서비스 개요

WavenetHelper은 조직 내 부서 간 업무 요청을 접수 → 배정 → 처리 → 확인까지 일관된 프로세스로 관리하는 시스템입니다. 데스크톱 위젯, 웹 대시보드, 외부 업체 포털의 3가지 채널을 제공하여 어디서든 업무를 관리할 수 있습니다.

🖥️

데스크톱 위젯

Windows 바탕화면에 항상 떠 있는 플로팅 위젯으로, 클릭 한 번에 업무 현황을 확인하고 새 업무를 요청할 수 있습니다.

🌐

웹 대시보드

브라우저에서 접속하는 풀 기능 대시보드입니다. 통계, 검색, 필터, 관리자 설정 등 모든 기능을 사용할 수 있습니다.

🏢

외부 업체 포털

거래처/협력사가 별도 계정 없이 전용 URL과 비밀번호로 업무를 요청하고 진행 상황을 확인할 수 있습니다.

데스크톱 위젯 (Electron)

Windows 바탕화면에 항상 떠 있는 플로팅 위젯입니다. 업무 현황을 실시간으로 확인하고, 클릭 한 번으로 상세 내역을 새 창에서 볼 수 있습니다.

1

플로팅 버튼 (접힌 상태)

바탕화면 모서리에 원형 버튼으로 표시됩니다. 미처리 업무 수량에 따라 색상과 크기가 자동으로 변합니다 — 마치 신호등처럼 긴급도를 직관적으로 파악할 수 있습니다.

0
0건 — 회색
3
1~5건 — 파랑
8
6~10건 — 주황
15
11~20건 — 빨강 (펄스)
25
20건+ — 진빨강 (펄스)
💡
펄스 애니메이션

11건 이상이면 버튼 주변에 파동 효과가 생기며, 미처리 업무가 쌓여 있음을 시각적으로 알려줍니다.

2

4방향 코너 스냅

위젯을 드래그하여 화면의 네 모서리 중 원하는 위치로 이동할 수 있습니다. 놓으면 가장 가까운 모서리에 자동으로 정렬됩니다.

↖️ 좌측 상단↗️ 우측 상단
↙️ 좌측 하단↘️ 우측 하단 (기본)

패널이 열릴 때 현재 코너 위치에 맞게 방향이 조정됩니다 (예: 좌측 하단이면 우측 위로 펼쳐짐).

3

확장 패널 (마우스 오버)

플로팅 버튼에 마우스를 올리면 다크 테마의 업무 목록 패널이 펼쳐집니다. 목록에는 대기중, 처리중, 확인요청 상태의 업무만 표시됩니다.

  • 업무 건수에 따라 패널 높이가 자동 조절 (최소 240px ~ 최대 700px)
  • 각 업무 항목에 상태 아이콘, 제목, 카테고리, 우선순위, 요청자, 경과시간 표시
  • 업무 클릭 시 별도 새 창으로 상세 내역 표시
  • 하단에 "새 업무 요청" 버튼
  • 마우스가 패널 밖으로 나가면 자동으로 접힘
  • 헤더 우측에 로그아웃 버튼
4

상세 창 & 업무 생성 창

업무를 클릭하거나 "새 업무 요청" 버튼을 누르면 별도 Electron 창이 열립니다. 이 창에서 업무 상세 내역 확인, 상태 변경, 댓글 작성, 파일 첨부 등 모든 작업이 가능합니다.

ADMIN

관리자 기능

관리자(admin) 역할을 가진 사용자만 접근 가능한 8가지 관리 탭을 제공합니다.

1

📊 운영 현황 대시보드

담당자별 처리 성과, 주간 업무 트렌드 (최근 4주), 외부 업체별 요청 통계, SLA 위반 목록을 종합적으로 확인합니다.

2

👥 사용자 관리

전체 사용자 목록 조회, 검색, 역할(admin/handler/user) 변경, 소속 부서 변경이 가능합니다. 역할별 인원 현황을 카드로 보여줍니다.

3

🏢 부서 관리

부서 생성/수정/삭제, 부서 색상 설정, SLA 기준 시간 설정(경고/긴급/에스컬레이션)이 가능합니다. 각 부서의 소속 인원 수도 표시됩니다.

4

🏭 외부 업체 관리

거래처/협력사 등록, 전용 URL(slug) 설정, 접속 비밀번호 관리, 대상 부서 및 기본 카테고리 설정, 활성/비활성 토글이 가능합니다.

5

⏰ 리마인더 설정

업무 단계별(접수대기/처리중/확인대기) 리마인더 주기를 설정합니다. 부서별, 카테고리별로 세분화된 리마인더 규칙을 만들 수 있습니다.

6

📜 활동 로그

전체 업무의 상태 변경 이력을 타임라인 형식으로 확인합니다. 기간별(오늘/이번주/이번달/전체), 상태별 필터링 및 페이지네이션을 지원합니다.

7

📥 데이터 내보내기

업무 목록 및 통계를 CSV 형식으로 다운로드합니다. 기간과 상태 필터를 적용할 수 있으며, Excel 호환을 위한 UTF-8 BOM이 포함됩니다.

8

🔧 시스템 설정

회사명, 업무 카테고리 목록 등 시스템 전반의 기본 설정을 관리합니다.

⚠️
관리자 전용

설정 탭은 관리자(admin) 역할을 가진 사용자에게만 웹 사이드바에 표시됩니다. 일반 사용자와 담당자에게는 보이지 않습니다.

EXTERNAL PORTAL

외부 업체 포털

외부 거래처/협력사가 별도 계정 없이 업무를 요청할 수 있는 전용 페이지입니다.

1

접속 방법

관리자가 등록한 업체별 전용 URL로 접속합니다.

// 접속 URL 형식
https://도메인/request/{업체슬러그}

// 예시
https://도메인/request/hankook   ← 한국물산
https://도메인/request/globaltech   ← 글로벌테크
2

인증

관리자가 설정한 비밀번호를 입력하여 인증합니다. 별도의 회원가입이나 이메일 계정이 필요하지 않습니다.

3

사용 가능 기능

  • 업무 요청 — 제목, 내용, 카테고리, 우선순위를 입력하여 요청
  • 요청 목록 — 해당 업체가 요청한 업무 목록 및 상태 확인
  • 상세 확인 — 처리 응답, 댓글 내용 확인
  • 댓글 — 담당자와 실시간 소통
  • 확인 응답 — 담당자의 확인 요청에 대해 응답
NOTIFICATIONS

알림 시스템

업무의 주요 이벤트가 발생할 때 관련 사용자에게 실시간으로 알림이 발송됩니다. Supabase Realtime을 통해 페이지 새로고침 없이 즉시 수신됩니다.

알림 종류 설명 수신 대상
새 업무 접수 새로운 업무 요청이 접수되었을 때 담당자
업무 배정 업무가 특정 담당자에게 배정되었을 때 담당자
확인 요청 담당자가 요청자에게 확인을 요청했을 때 요청자
처리 완료 업무가 완료 처리되었을 때 요청자
SLA 초과 업무 처리 시간이 SLA 기준을 초과했을 때 담당자/관리자
리마인더 설정된 주기에 따라 미처리 업무를 알려줄 때 담당자
재배정 관리자가 다른 담당자에게 업무를 재배정했을 때 새 담당자
⚡
실시간 동기화

Supabase Realtime을 통해 업무 상태 변경, 댓글 추가, 알림 수신이 페이지 새로고침 없이 자동으로 반영됩니다. 데스크톱 위젯과 웹 대시보드 모두 동일합니다.
